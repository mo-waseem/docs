# BridgeAI Features Overview

BridgeAI provides a complete ecosystem for managing leads, conversations, automation flows, properties, appointments, and CRM integrations — all designed for real estate agencies that need efficiency, consistency, and speed.

This section gives you a structured overview of every feature in the platform, what it does, and where to find the detailed documentation.
